# Security Policy

## Supported Versions

Currently, only the latest release of Terminal Setup Style is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| > 0.1.x | :white_check_mark: |
| < 0.1.x | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Terminal Setup Style, please send an e-mail to Angel-T Dev via the GitHub repository's contact options or create a private vulnerability report. All security vulnerabilities will be promptly addressed.
