## вңЁ CaracterГӯsticas

* **InstalaciГіn de 1 Clic:** OlvГӯdate de configuraciones largas en PowerShell. La extensiГіn hace todo el trabajo pesado.
* **Motor Oh My Posh:** Inyecta un diseГұo visual moderno y fГЎcil de leer directamente en tu editor.
* **IconografГӯa Avanzada:** Soporte total para iconos de carpetas, estado de Git y lenguajes de programaciГіn.
* **IntegraciГіn Nativa:** Modifica automГЎticamente el `settings.json` de VS Code para aplicar la fuente correcta sin que tengas que buscarla.

## рҹ“Ӣ Requisitos Previos

ВЎBuenas noticias! A partir de la versiГіn **0.0.7**, la extensiГіn **descarga e instala automГЎticamente** las fuentes necesarias (*FiraCode Nerd Font* y *CaskaydiaCove Nerd Font*) en tu perfil de usuario (sin requerir permisos de administrador). 

*Nota: Solo si experimentas algГәn fallo en la descarga automГЎtica, puedes instalarlas manualmente desde la pГЎgina de [Nerd Fonts](https://www.nerdfonts.com/).*

## рҹҡҖ CГіmo Usarla

1. Abre Visual Studio Code.
2. Abre la Paleta de Comandos presionando `Ctrl` + `Shift` + `P`.
3. Escribe y selecciona el comando: **`Angel-T Dev: Instalar Terminal Setup`**.
4. ВЎEspera el mensaje de Г©xito, abre una nueva terminal y disfruta tu nuevo entorno!

## вҡҷпёҸ PersonalizaciГіn (ConfiguraciГіn)

Esta extensiГіn utiliza la interfaz nativa de configuraciГіn de Visual Studio Code para que puedas modificar tus parГЎmetros de forma integrada y sin complicaciones:

1. Abre la configuraciГіn de VS Code presionando `Ctrl` + `,` (o ve a *Archivo > Preferencias > ConfiguraciГіn*).
2. En la barra de bГәsqueda superior, escribe: **`Terminal Setup Style`**.
3. VerГЎs los campos de personalizaciГіn listos para ser editados:
   * **Nombre**: El nombre que se mostrarГЎ en tu prompt de PowerShell.
   * **Fuente**: El nombre de tu Nerd Font instalada.
   * **Tema**: MenГә desplegable para seleccionar tu tema de Oh My Posh favorito.
   * **Color ASCII**: MenГә desplegable para elegir el color del Arte ASCII de bienvenida (usa 'auto' para usar el del tema).
4. Modifica los valores. Se guardarГЎn automГЎticamente de forma global en tu VS Code.
5. Abre la Paleta de Comandos (`Ctrl` + `Shift` + `P`) y ejecuta **`Angel-T Dev: Instalar Terminal Setup`** para aplicar e instalar la terminal con tu nueva configuraciГіn.
